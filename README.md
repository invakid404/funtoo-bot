# funtoo-bot

> A bot to facilitate the Funtoo development process on GitHub

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Docker

```sh
docker run -e APP_ID=<app-id> -e PRIVATE_KEY=<pem-value> ghcr.io/invakid404/funtoo-bot
```
